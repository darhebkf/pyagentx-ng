# Introduction

**snmpkit** is a high-performance SNMP toolkit for Python, powered by a Rust core.

## Why snmpkit?

- **Fast** — Rust-powered core for maximum performance
- **Async-first** — Built on Python's asyncio
- **Type-safe** — Full type hints throughout
- **Simple API** — Clean, intuitive interface

## What can you build?

- SNMP agents that expose custom MIBs
- Network monitoring tools
- Device management systems
- Telemetry collectors

## Quick Example

```python
from snmpkit import Agent, ObjectType, OctetString

agent = Agent()

@agent.register("1.3.6.1.4.1.12345.1.1.0")
async def system_description():
    return ObjectType(OctetString("My SNMP Agent"))

agent.run()
```
