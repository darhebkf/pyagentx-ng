## Requirements

- Python 3.10+
- Net-SNMP (for AgentX support)

## Your First Agent

Create a simple SNMP agent:

```python filename="agent.py"
from snmpkit import Agent

agent = Agent()

@agent.register("1.3.6.1.4.1.12345.1.1.0")
async def get_uptime():
    return 42

if __name__ == "__main__":
    agent.run()
```

Run it:

## Configuration

snmpkit can be configured via environment variables or programmatically:

```python
from snmpkit import Agent

agent = Agent(
    master_socket="/var/agentx/master",
    name="my-agent",
    description="My Custom Agent",
)
```
