# Features

snmpkit provides a modern, performant approach to SNMP in Python.

## AgentX Protocol

Full implementation of RFC 2741 (AgentX protocol):

- Register OID subtrees with a master agent
- Handle GET, GETNEXT, GETBULK requests
- Support for SET operations
- Index allocation and context support

```python
from snmpkit import Agent

agent = Agent(master_socket="/var/agentx/master")

@agent.register("1.3.6.1.4.1.12345")
async def handle_subtree(oid):
    # Handle requests for entire subtree
    pass
```

## Async-First Design

Built on Python's asyncio for non-blocking operations:

```python
import asyncio
from snmpkit import Agent

agent = Agent()

@agent.register("1.3.6.1.4.1.12345.1.0")
async def get_data():
    # Non-blocking I/O
    data = await fetch_from_database()
    return data
```

## Rust Core

Performance-critical components written in Rust:

- PDU encoding/decoding
- OID parsing and trie operations
- Binary protocol handling

This gives you Python's ease of use with Rust's performance.

## Type Safety

Full type hints for better IDE support and fewer bugs:

```python
from snmpkit import Agent, ObjectType, Integer32

agent = Agent()

@agent.register("1.3.6.1.4.1.12345.1.0")
async def typed_handler() -> ObjectType[Integer32]:
    return ObjectType(Integer32(42))
```

## SNMP Types

Support for all standard SNMP types:

| Type | Description |
|------|-------------|
| `Integer32` | 32-bit signed integer |
| `Counter32` | 32-bit counter |
| `Counter64` | 64-bit counter |
| `Gauge32` | 32-bit gauge |
| `TimeTicks` | Time in hundredths of a second |
| `OctetString` | Arbitrary bytes |
| `ObjectIdentifier` | OID value |
| `IpAddress` | IPv4 address |
